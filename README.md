# tatoil-front
Front-end repo for project with tatoil

Для работы необходимо:
- node.JS (выполнение сборки)
- npm (менеджер пакетов)

Подготовка и сборка исходников:
`npm install`
`npm run build`
Собирает статику, которую можно напрямую использовать статическим сервером или динамическим сервером для отдачи статики.

Деевелоперская сборка:
`npm install`
`npm run dev`
Запускает локальный webpack-dev-server и разворачивает на свободном порту сервер в watch режиме, позволяя разрабатывать и дебажить фронт.
